# Indian Prison Stats - 2019

<h3>Overview</h3>

This project analyses statistics on inmates in Indian prisons for the year 2019 segregated on the basis of gender and state. Some of the metrics being considered are -
- \# convicts
- \# undertials
- \# detainees
- conviction %
